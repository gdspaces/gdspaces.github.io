Explanation: To configure an Azure web app to host a custom domain, the first step is to create a TXT record for the domain with the web app's default hostname as the value. In this case, the domain is www.contoso.com, and the web app's default hostname is contoso.azurewebsites.net. This TXT record is used for domain verification, which proves that you own the domain and have the right to use it with the web app. After the TXT record is created and verified, you can then configure the custom domain in the Azure portal and update the DNS records for the domain to point to the web app.
