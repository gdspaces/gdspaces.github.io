A. From the Licenses blade of Azure AD, assign a license. To enable users to use all the Azure AD Premium features, you need to assign the purchased Azure AD Premium P2 licenses to 10 users in the tenant. This can be done from the Licenses blade of Azure AD. From there, you can select the 10 users to assign the licenses to and assign them the Azure AD Premium P2 licenses. This will enable them to use all the Azure AD Premium features.
