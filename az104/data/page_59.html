You have an Azure Directory (Azure AD) tenant named Adatum and an Azure Subscription named Subscription1. Adatum contains a group named Developers.
Subscription1 contains a resource group named Dev.
You need to provide the Developers group with the ability to create Azure logic apps in the Dev resource group.
Solution: On Subscription1, you assign the Logic App Operator role to the Developers group.
Does this meet the goal?

  • A. Yes
  • B. No

Correct Answer: B You would need the Logic App Contributor role. Logic App Operator - Lets you read, enable, and disable logic apps, but not edit or update them.
