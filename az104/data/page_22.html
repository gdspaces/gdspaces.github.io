You plan to deploy several new virtual machines (VMs) in Azure. The VMs will have the same operating system and custom software requirements.

Which PowerShell cmdlets should you use?

  • B. Add-AzVhd

  • D. Add-AzImageDataDisk

2¬†months, 2¬†weeks ago
Selected Answer: C
To upload a generalized virtual machine (VM) image to Azure for use in creating new Azure VMs, the PowerShell cmdlets that should be used are: 1. Sysprep the reference VM in the on-premises virtual environment to generalize the image. 2. Use the Export-AzVM cmdlet to export the generalized VM to a .zip file. 3. Use the New-AzStorageAccount cmdlet to create an Azure storage account. 4. Use the Set-AzStorageBlobContent cmdlet to upload the .zip file to the Azure storage account. 5. Use the Add-AzImage cmdlet to create a new custom image from the uploaded .zip file. Therefore, the correct answer is C. Add-AzImage. Note that Add-AzVM, Add-AzVhd, and Add-AzImageDataDisk are not the appropriate cmdlets for uploading a generalized VM image to Azure. Add-AzVM is used for creating new virtual machines in Azure, while Add-AzVhd and Add-AzImageDataDisk are used for attaching virtual hard disks and data disks to virtual machines.
