NO- User2 has only Virtual Machine User Login on Sub1( Sub1/RG1/VM1). Virtual Machine User Login- View Virtual Machines in the portal and login as a regular user. Only read the Public IP addresses ,Virtual Networks, LoadBalancers, Network Interfaces, VMs, Hybrid Compute/machines and list credentials for endpoints.
