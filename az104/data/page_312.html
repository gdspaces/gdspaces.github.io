  • C. a frontend IP configuration
  • D. a load balancing rule

Selected Answer: A
A. an inbound NAT rule. To direct all RDP connections to VM3 only, you need to create an inbound NAT rule that maps the RDP port (3389) to the RDP port of VM3. You can do this by specifying the frontend IP configuration of the public load balancer, the protocol (TCP), the frontend port (3389), and the backend port (3389) of VM3 in the inbound NAT rule. This will route all incoming RDP traffic to VM3 only, regardless of the load balancing configuration.
