Create an Azure Active Directory (Azure AD) security group that includes the users or groups that you want to grant access to the file share. 2. Create a file share in Azure Files or use an existing file share. 3. Navigate to the file share and select "Access control (IAM)" from the left-hand navigation menu. 4. Click on the "Add" button and select "Add role assignment." 5. In the "Add role assignment" pane, select the role that you want to assign to the security group. For example, "Storage File Data SMB Share Contributor" grants read and write access to the file share. 6. Select the security group that you created in step 1. 7. Click on the "Save" button to create the role assignment. Once you have completed these steps, the members of the security group will have access to the file share according to the role that you assigned to them. They can access the file share using the SMB protocol, Azure Files REST API, or Azure File Sync.
